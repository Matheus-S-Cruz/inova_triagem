# Triagem+

Sistema de pré-triagem para orientar cidadãos sobre qual tipo de unidade de saúde procurar (UBS, UPA ou Hospital), com base nos sintomas informados.

> **Importante:** O sistema não realiza diagnósticos médicos e não substitui a avaliação de um profissional da saúde. Seu objetivo é apenas orientar o usuário para o local de atendimento mais adequado.

---

## Objetivo

Reduzir a superlotação das unidades de saúde direcionando os pacientes para o serviço mais adequado às suas necessidades.

### Benefícios

- Atendimento mais rápido;
- Redução de filas;
- Melhor distribuição dos pacientes;
- Facilidade de acesso à informação.

## Como executar o projeto

### 1. Clone o repositório

```bash
git clone <url-do-repositorio>
```

### 2. Acesse a pasta

```bash
cd triagem-plus
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Execute o projeto

```bash
npm run dev
```

O projeto estará disponível em:

```
http://localhost:5173
```
